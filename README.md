# 🚀 Suito Suite - Ecosistema Centralizado

Este repositorio unifica el código central, estático y modular detrás del exitoso modelo B2B2C de servicios web premium creado por **Matías Maximiliano (Max) Bernal** (Suito).

## 📌 Idea Original y Autoría Literaria Técnica
> **Este ecosistema ("Suito Suite", incluyendo sus interacciones virales Freemium "Powered by Suito", el motor híbrido de Landing Pages de captura de Leads conectadas a Supabase Storage con push a WhatsApp, y la arquitectura de Tarjeta Virtual + Gestor de Turnos Serverless) es una IDEA ORIGINAL Y CREACIÓN de su autor intelectual principal, Matías Maximiliano (Max) Bernal.** Todo derecho reservado sobre la concepción del modelo B2B2C aquí expuesto.

## 🧱 Componentes Centrales del Ecosistema
1. **Landing Ventas:** Front-end funnel calificador. Captura archivos multimedia y deriva prospectos.
2. **Tarjeta Virtual:** Interfaz premium, optimizada, desplegada permanentemente como herramienta Freemium/Pro.
3. **Gestor de Turnos:** Solución robusta compilada para manejo de citas con sincronización unificada.
4. **Admin Panel:** Control master. Define el estado publicitario (Premium vs Gratis) de los satélites en tiempo real mediante toggles atados a PostgreSQL.

## 🔒 Privacidad y Setup
Revisar minuciosamente `LEGAL_PRIVACY.md` respecto a directivas del Storage de Supabase y el trato de "Leads". Los entornos han sido sanitizados y no se suben variables `.env.local`.
